# Analysis Workspace

Use this directory for Python and MATLAB post-test analysis.

First useful script:

- Load a CSV from `data/raw/`.
- Plot `roll_deg`, `pitch_deg`, `pid_roll`, `pid_pitch`, and servo commands.
- Report loop timing statistics from `loop_dt_us`.

Keep raw SD logs unchanged in `data/raw/`; write cleaned or derived outputs to `data/processed/`.
